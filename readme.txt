Para criar e inicializar o projeto nodejs api


abrir o terminal/command
criar a pasta com o nome do projeto
entrar na pasta com o nome do projeto
digitar e executar o comando

npm init
confirma as informações nescessarias de acordo com os requisitos


apos esse comando o arquivo package.json sera criado

npm install --save bcrypt cors dotenv express nodemon mysql2 sequelize
